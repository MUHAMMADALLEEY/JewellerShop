# Madina Jewellers — 3D Showcase Site

A dark-luxury, 3D-driven brand showcase for **Madina Jewellers**, the goldsmith atelier of **Muhammad Arshad**. Built with Next.js 14, React Three Fiber, GSAP, Framer Motion and Tailwind CSS.

## What it includes

- **Hero** — full-viewport 3D scene with a rotating gold ring, orbiting diamonds, pavé band detail, sparkle field and mouse-driven camera drift
- **Collection** — animated masonry gallery with category filter
- **Atelier** — interactive 3D product viewer (rotate, zoom, switch pieces)
- **Heritage** — parallax craftsman story with a floating 3D gem
- **The Signature** — dedicated section featuring Muhammad Arshad with a halo-ring 3D scene
- **Contact** — validated enquiry form (react-hook-form + zod)

## Stack

| | |
|---|---|
| Framework | Next.js 14 (App Router, TypeScript) |
| Styling | Tailwind CSS |
| 3D | three, @react-three/fiber, @react-three/drei, @react-three/postprocessing |
| Motion | framer-motion, gsap, lenis |
| Forms | react-hook-form, zod |

## Run locally

```bash
npm install
npm run dev
```

Then open [http://localhost:3000](http://localhost:3000).

## Design

- Palette: noir `#0a0a0a` · gold `#d4af37` · cream `#f5ecd9`
- Display type: Cormorant Garamond · Body: Inter
- Custom gold-dot cursor, gold-dust particle layer, smooth scroll via Lenis
